# Tulis Izin Digital 🖋️

![Vercel Deployment](https://deploy-badge.vercel.app/api/deploy-badge?url=https://github.com/rizd027/Tulis-Digital)

A premium, web-based digital letter application designed for creating permission letters (Surat Izin) quickly and elegantly. Features a modern dashboard, real-time paper preview, and various customization options.

## Features ✨

*   **Premium Editor**: Modern toolbar with advanced formatting (lists, alignment, basic styles).
*   **Real-time Preview**: See your letter exactly as it will look on paper with high-fidelity rendering.
*   **Mobile-First Design**: Optimized for mobile with floating navigation and minimalist controls.
*   **Customization**: Change fonts, paper types (Folio/A4), and layout settings instantly.
*   **Export Options**: Download as image or print directly.
*   **Support Developer**: Integrated Saweria link for direct support.

## Tech Stack 🛠️

*   **Next.js 14** (App Router)
*   **Tailwind CSS**
*   **Framer Motion** (Animations)
*   **Lucide React** (Icons)

## Development

npm install
npm run dev
